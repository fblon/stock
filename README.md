# Stock Tracker Project
## Startup

`npm install`\
`npm start`

http://localhost:4200/

## Tests

`npm run cypress:headless`

## Sanity

`npm run lint`

## Deployment

`npm run buildPages`\
`git add .`\
`git commit -am 'update git pages'`

Site automatically deployed by github and accessible at: https://fblon.github.io/stock-tracker/
